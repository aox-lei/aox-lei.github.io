### 1. 输出第几行
$0 代表整行
```
awk '{print $1}' # 输出第一行
```

### 2. 过滤数据
```
awk '$1==1' # 第一列等于1的数据
```
其他计算符号 __!=、>、<、>=、<=__
```
# 过滤并输出指定列
awk '$1==1 {print $1}'
```

### 3. 格式化输出
```
awk '{printf "%-8s %-8s %-8s %-18s %-22s %-15s\n",$1,$2,$3,$4,$5,$6}' netstat.txt
```

### 4. 指定分隔符和指定输出分隔符
```
awk -F 分隔符 -OFS 输出的分隔符
```

### 5. 字符串查找
模式取反用!~
```
awk -F : '$1 ~ /www/ {print $1  $2}' /etc/passwd
```

### 6. 拆分文件
```
awk -F : '$1 ~ /www/ {print > $2}' /etc/passwd
```
